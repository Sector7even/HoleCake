
HoleCake ist ein RAT (Remote Access Tool), mit dem andere Computer über eine 
Verbindung fernüberwacht und gesteuert werden können über die Windows Shell.
Über die Shell(CMD) in Windows lassen sich viele Aufgaben schneller abarbeiten 
als mit einer grafischen Benutzeroberfläche (GUI). Des Weiteren funktionieren 
bestimmte Befehle und Programme in Windows nur per Kommandozeile.



Mittels HoleCake kann man auf einem entfernten Rechner:

    Dateien löschen, herauf- und herunterladen z.b(FTP)
    Programme ausführen
    Konfigurationen ändern
    Aktionen ausführen, wie beispielsweise Rechner neu starten, 
    beliebige Dialogfenster einblenden z.b(MSG)
